总结记录一下python装饰器和单例的实现

一、装饰器模式

装饰器由闭包实现，闭包概念如下：

  • 绑定外部作用域的变量的函数
  • 即使离开外部作用域，如果闭包仍可见，绑定变量不会销毁
  • 每次运行外部函数都会重新创建闭包
